# 我的 Ghostty 設定

在 MacOS 上，使用快捷鍵 `Cmd + ,` 可以快速開啟用來設定 Ghostty 的設定檔。

```text
theme = Catppuccin Frappe
font-family = CaskaydiaMono Nerd Font
font-family = Noto Sans TC
font-size = 22

window-padding-x = 20
window-padding-y = 10

adjust-cell-height = 18

background-image = /full/path/to/image.jpg
background-image-opacity = 0.05
background-image-fit = cover
```

調整完設定後，可以使用 `Shift + Cmd + ,` 來重新載入設定。

## Theme

Ghostty 支援多種主題，並且與 [Iterm2-color-schemes](https://iterm2colorschemes.com/) 同步，
可以在設定檔中使用 `theme` 來切換。
